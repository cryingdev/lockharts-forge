# Lockhart's Forge - Combat Formula Specification

이 문서는 게임 내 전투 계산의 최종 순서와 공식을 정의합니다. 모든 전투 로직은 이 명세서를 기준으로 구현되어야 합니다.

---

## 1. 파생 스탯 계산 (Derived Stats)
기본 스탯(Primary Stats)과 레벨을 기반으로 전투에 직접 사용되는 파생 스탯을 산출합니다.
기준 파일: `config/derived-stats-config.ts`

### 1.1 공격 및 방어
- **물리 공격력 (ATK)**: `(Level * 1.5) + (STR * 2.5) + (DEX * 0.2)`
- **마법 공격력 (MATK)**: `(Level * 1.0) + (INT * 2.5)`
- **물리 방어력 (DEF)**: `(Level * 0.8) + (VIT * 2.2)`
- **마법 방어력 (MDEF)**: `(Level * 0.5) + (INT * 1.0) + (VIT * 1.4)`

### 1.2 피해 감소율 (Reduction)
방어력을 실제 피해 감소 비율(0~1)로 변환합니다.
- **감소율 공식**: `Defense / (Defense + 150)`
- *예: 방어력이 150이면 감소율은 0.5 (50%)*

### 1.3 명중, 회피, 속도
- **명중 (ACC)**: `70 + (DEX * 1.2) + (LUK * 1.0)`
- **회피 (EVA)**: `5 + (DEX * 1.2) + (LUK * 0.8)`
- **속도 (SPD)**: `100 + (DEX * 2.0)`

### 1.4 치명타 (Critical)
- **치명타 확률 (Crit %)**: `5 + (LUK^0.85 * 2.0) + (DEX * 0.25)` (최대 80%)
- **치명타 피해 (Crit Dmg %)**: `150 + (LUK * 1.5)` (최대 350%)

## 2. 전투 판정 순서 (Combat Calculation Sequence)
전투 발생 시 다음 순서에 따라 최종 데미지를 결정합니다.
기준 파일: `utils/combatLogic.ts`

### Step 1: 명중 판정 (Hit Check)
공격자의 명중(ACC)과 방어자의 회피(EVA)를 비교합니다.
1. **기본 명중률**: `(ACC / (ACC + EVA * 0.7)) * 100`
2. **보정**: 최소 90%, 최대 100%로 고정 (Clamped)
3. **결과**: 난수(0~100)가 명중률보다 크면 **MISS** (데미지 0).

### Step 2: 숙련도 변환 (Efficiency Transformation)
공격자의 직업별 숙련도(Job Efficiency)를 적용하여 공격의 안정성을 결정합니다.
1. **최소 숙련도**: 직업별 설정값 (예: Fighter 85%)
2. **변환 숙련도**: `Random(최소 숙련도, 100) / 100`

### Step 3: 유효 공격력 산출 (Effective Attack)
1. **공격력**: 물리 또는 마법 공격력 선택
2. **유효 공격력**: `공격력 * 변환 숙련도 * 스킬 배율`

### Step 4: 방어력 적용 (Defense Reduction)
1. **피해량**: `유효 공격력 * (1 - 방어자의 감소율)`

### Step 5: 치명타 판정 (Critical Hit)
1. **판정**: 난수(0~100)가 공격자의 치명타 확률보다 작거나 같으면 **CRIT**.
2. **배율 적용**: `피해량 * (치명타 피해 / 100)`

### Step 6: 최종 데미지 확정
1. **최종 데미지**: `Max(1, Round(피해량))`
- 최소 1의 데미지는 항상 보장됩니다.

---

## 3. 전투력 지수 (Combat Power - CP)
캐릭터의 종합적인 강함을 나타내는 지표입니다.

- **공격 점수**: `기대 DPS * 12`
- **방어 점수**: `(최대 HP / (1 - 평균 감소율)) * (1 + 회피 * 0.005) * 0.85`
- **최종 CP**: `(공격 점수 + 방어 점수) * 0.1` (반올림)
